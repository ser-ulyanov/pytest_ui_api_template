# pytest_ui_api_template

## Шаблон для автоматизации тестирования на python

### Шаги
1. Склоинировать проект 'git clone https://github.com/ser-ulyanov/pytest_ui_api_template.git'
2. Установить все зависимости  
3. Закпустить тесты 'pytest'
### Стек:
- pytest
- selenium
- requests
- _sqlalchemy_
- allure
- config

### Структура
- ./test - тесты
- ./pages - описание страниц
- ./api - хелперы для работе с API
- ./db - хелперы для работе с БД
